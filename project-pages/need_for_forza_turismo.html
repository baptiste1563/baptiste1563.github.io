Need For Forza Turismo
Jeu type Endless Runner créer sur Unity 3D. Avec routes et obstacles a génération procédurale, plusieurs modes de jeux et vue cockpit fonctionnelle.
Détail sur le projet :
Dans ce petit jeu de style Endless runner vous conduisez une petite camionette qui esquive les obstacle afin d'aller le plus loin possible
Il a été réaliser sur Unity 3D, j'ai notament travailler sur les fonctionnalité telle que : la vue cockpit, la physique du véhicule ou encore la génération aléatoire de la route
